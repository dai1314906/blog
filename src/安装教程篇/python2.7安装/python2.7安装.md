# Python2.7安装

一、python官网地址：

https://www.python.org/downloads/release/python-270/

![下载python安装文件](.\img\1.png)

二、安装python

1、双击启动`python-2.7.amd64.msi`文件

2、打开后选择Install for all users,然后点击 Next 进入下一步

![python安装](.\img\2.png)

3、现在选择Python 2.7安装的位置，默认是C:\Python27，这里可以根据自己的需求选择安装目录，但是目录名中最好不要包含中文.
如果C盘空间大，可以不更改，或有固态硬盘 的，可以选择将Python 2.7装在固态盘中，确认好安装位置后,点击 Next 进入下一步

![python安装](.\img\3.png)

4、后续一直next就行，最后点击finish完成就行

![python安装](.\img\4.png)

5、验证是否安装成功。
